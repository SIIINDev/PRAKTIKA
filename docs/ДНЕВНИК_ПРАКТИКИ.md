# Дневник учебной практики

**Тема:** Разработка интеллектуальной поисковой системы по внутренней базе знаний университета.
**Направление:** 09.03.04 «Программная инженерия».
**Срок:** 23 июня 2026 г. — 06 июля 2026 г. (12 рабочих дней календарного графика).
**Команда:** 4 человека — BE (Backend), FE (Frontend), DO (DevOps), QA.

Записи ведутся по дням; в каждом дне фиксируется общекомандная задача и индивидуальная работа
по ролям с привязкой к реализованным функциям и файлам.

---

## Итог

К завершению практики (06 июля 2026) система запускается одной командой `docker compose up --build`,
выполняет полный цикл «загрузка → индексация → поиск → подсветка → пагинация», покрыта юнит- и
E2E-тестами, снабжена мониторингом и полной документацией. Распределение работ по ролям и
выполнение всех требований зафиксированы в `docs/REQUIREMENTS_TRACEABILITY.md`.
